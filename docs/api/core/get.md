# Download Module

The `geefetch.data.get` module contains the primary functions for initiating and managing satellite data downloads from Google Earth Engine.

::: geefetch.data.get

    options:
        show_root_heading: true
        show_source: true
        heading_level: 3
        members_order: source
        filters:
        - "!^_"
        - "download.*"
        - "!download_chip.*"
